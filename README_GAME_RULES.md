# Chickenmaster2 - 게임 규칙

## 게임 목표

한국 치킨집을 경영하는 시뮬레이션 웹 게임입니다.
매일 준비 → 영업 → 야간 활동 → 수면의 4구간을 반복하며 매출을 올리고 가게를 성장시킵니다.

---

## 하루 흐름 (4구간 시스템)

1턴 = 1일, 24시간을 4구간으로 나누어 진행합니다.

| 구간 | 시간대 | 조절 범위 | 플레이 방식 |
|------|--------|-----------|-------------|
| **PREP** (준비) | 기상~오픈 | wake(5-8), open(9-12) | 카드 큐잉 → 확인 |
| **BUSINESS** (영업) | 오픈~마감 | open(9-12), close(18-23) | 타임랩스 + 의사결정 |
| **NIGHT** (야간) | 마감~취침 | close(18-23), sleep(22-26) | 카드 큐잉 → 확인 |
| **SLEEP** (수면) | 취침~기상 | 자동 계산 | 자동 피로 회복 |

기본값: wake=7, open=10, close=21, sleep=24
→ PREP 3h, BUSINESS 11h, NIGHT 3h, SLEEP 7h

## 행동 카드

### 준비 구간 (PREP)

| 행동 | 시간 | 비용 | 효과 |
|------|------|------|------|
| 재료 준비 | 1h | - | prepared_qty +5, 재료 -3 |
| 재료 점검 | 1h | - | 요리+4, 경영+2 |
| 매장 청소 | 2h | - | 서비스+10 |
| 장비 점검 | 1h | - | 기술+8, 경영+4 |
| 휴식 | 0.5h | - | 피로 회복 |

### 야간 구간 (NIGHT)

| 행동 | 시간 | 비용 | 효과 |
|------|------|------|------|
| 레시피 연구 | 3h | - | 요리+15 |
| 경영 연구 | 3h | - | 경영+15 |
| 광고 연구 | 3h | - | 경영+12, 기술+8 |
| 서비스 연구 | 3h | - | 서비스+15 |
| 학습 | 2h | - | 기술+15, 경영+5 |
| 운동 | 2h | - | 체력+20 |
| 전단지 배포 | 2h | 5만원 | 경영+6 |
| 온라인 광고 | 1h | 10만원 | 경영+8, 기술+4 |
| 배달앱 등록 | 1h | 3만원 | 경영+6, 기술+6 |
| 재료 주문 | 1h | 20만원 | 재료 +50 |
| 알바 고용 | 2h | 8만원 | 경영+10, 서비스+4 |
| 휴식 | 0.5h | - | 피로 회복 |

남는 시간은 자동으로 휴식(0.5h 단위)으로 채워집니다.

## 영업 시스템 (BUSINESS)

영업 구간은 타임랩스로 진행됩니다 (2초 = 1시간).

### 고객 수 계산
```
기본 고객수 = 영업시간 × 3 × (평판 / 50)
```

### 의사결정 이벤트
영업 중 1~2회 랜덤 이벤트가 발생합니다.
각 이벤트에서 A/B 중 하나를 선택하면 즉시 효과가 적용됩니다.

예시:
- "점심 러시! 할인할까요?" → 할인(고객+30%, 마진-20%) / 정가 유지
- "단체 예약 15명" → 수락(+15명, 재고 소모) / 거절
- "서비스 컴플레인" → 정중 대응(평판+5) / 무시(평판-5)

### 판매 계산
```
판매 가능량 = 준비된 수량 + 재고
실제 판매 = min(총 고객수, 판매 가능량)
매출 = 실제 판매 × 유효 가격
```
준비된 수량을 먼저 소진하고, 부족분을 재고에서 차감합니다.

## 수면 & 피로

- 수면 시간당 피로 회복: **16**
- 기본 수면 7시간 → 112 회복
- 피로도 0 미만으로 내려가지 않음
