# Generated Configs

This folder stores experiment configuration files generated by AI agents.

## Rules

1. **Never directly modify default config files (`configs/default.yaml`, etc.)**
2. All experiment-specific configs should be created as new files in this folder.
3. Naming convention: `YYYY-MM-DD_<experiment_name>.yaml`

## Example

```yaml
# configs/generated/2024-12-02_lr_test.yaml
experiment:
  name: "lr_test_001"
  base_config: "../default.yaml"  # Reference to default config

# Only specify changed parameters
training:
  learning_rate: 2e-4
```

## Cleanup

- Config files no longer needed after experiments can be periodically cleaned up.
- Important experiment configs are preserved in result JSON files, so deletion is safe.
