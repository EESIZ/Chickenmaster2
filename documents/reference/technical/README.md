# Technical Notes

This folder stores reusable technical knowledge that should be easy to search and reuse.

## Conventions

- **Filename**: `NOTE_<topic>.md`
- **Language**: Korean for narrative, English for code/config.
- **Cross-links**: Link to related code, configs, results, and other docs.

## Optional Template

If you want a starting structure, use the archived template:

- `documents/archive/templates/TEMPLATE_NOTE.md`
